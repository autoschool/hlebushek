hlebushek
=========
### Run application ###
clone project repository to local:
```
$git clone https://github.com/autoschool/hlebushek.git
```
go to project directory:
$cd hlebushek
run server:
```
$mvn jetty:run
```
open in browser:
```
http://localhost:8080
```

### Create branch ###
clone project repository to local:
```
$git clone https://github.com/autoschool/hlebushek.git
```
go to project directory
```
$cd hlebushek
```
create new branch:
```
$git branch [branch_name]
```
change current branch:
```
$git checkout [branch_name]
```
